Project to train different gpt-like models using context free grammars. This is based on work from the paper [Physics of Language Models: Part 1, Learning Hierarchical Language Structures](https://arxiv.org/abs/2305.13673)

## Setup

First install the CFG library found here: https://github.com/PolarState/CFG. Clone it and install the package locally (preferably in a virtual environment) via:

```bash
git clone git@github.com:PolarState/CFG.git
cd CFG
python -m pip install -e .
```

## Usage

Training is launched via `main.py` with the `-m` flag to select a model architecture.

### GPT2

```bash
python main.py -m GPT2
```

Uses the default GPT2 config and the `gpt2` tokenizer from HuggingFace. Training data is loaded from pre-tokenized binary datasets generated by the CFG library.

### GPTNeoX

```bash
python main.py -m GPTNeoX
```

Uses a GPTNeoX config sized to match GPT2 (768 hidden, 12 layers, 12 heads) but with rotary position embeddings instead of learned embeddings. Uses the GPT2 tokenizer via `GPTNeoXTokenizerFast`.

### Resuming training

To resume from the latest checkpoint:

```bash
python main.py -m GPTNeoX --resume
```

### Validating checkpoints

`generate.py` loads saved checkpoints, generates samples, and validates them against the grammar:

```bash
python generate.py -m GPTNeoX
```
